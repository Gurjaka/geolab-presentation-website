# Project Showcase Hub: Movie Recommendation System

ეს გვერდი წარმოადგენს პროექტის ცენტრალურ ჰაბს, რომელიც შეიქმნა პრეზენტაციის პროცესის ოპტიმიზაციისა და ტექნიკური დოკუმენტაციის გამჭვირვალობისთვის.

---

## გვერდის მიზანი

ამ ვებგვერდის მთავარი ამოცანაა, ერთ სივრცეში გააერთიანოს პროექტის ვიზუალური და ტექნიკური მხარე:

1.  **პრეზენტაციის დემონსტრირება:** ინტერაქტიული სლაიდების ნახვა პირდაპირ ბრაუზერში.

2.  **ცოდნის გაზიარება:** პრეზენტაციის დასრულების შემდეგ, ნებისმიერ დაინტერესებულ პირს აქვს წვდომა პროექტის დეტალურ ტექნიკურ აღწერაზე.

3.  **რეალურ დროში განახლება:** გვერდი დინამიკურად კითხულობს `README.md` ფაილს, რაც უზრუნველყოფს დოკუმენტაციის მუდმივ აქტუალობას.

---

## როგორ არის აწყობილი (Tech Stack)

გვერდი აგებულია "Simplicity & Performance" პრინციპით:

* **HTML5 & CSS3:** მინიმალისტური და სუფთა ვიზუალისთვის.

* **JavaScript (Fetch API):** მონაცემთა ასინქრონული წამოღება რეპოზიტორიიდან.

* **Marked.js:** ბიბლიოთეკა, რომელიც Markdown ფორმატს გარდაქმნის სუფთა HTML-ად რეალურ დროში.

* **Google Slides API:** პრეზენტაციის სლაიდების ინტეგრაცია `iframe`-ით.

## სტრუქტურა

1.  **Header:** პროექტის სათაური და დეველოპერის ინფორმაცია.

2.  **Presentation Deck:** სლაიდების სექცია, სადაც შესაძლებელია პრეზენტაციის სრულეკრანიან რეჟიმში ნახვა.

3.  **Markdown Documentation:** ქვედა სექცია, სადაც ავტომატურად რენდერდება პროექტის ტექნიკური დეტალები, ინსტალაციის ინსტრუქცია და არქიტექტურული გადაწყვეტილებები.

## რატომ არის ეს მნიშვნელოვანი დეველოპერისთვის?

როგორც დეველოპერი, მე მჯერა, რომ კარგი კოდი საჭიროებს კარგ დოკუმენტაციას. ეს ვებ-გვერდი არის იმის დასტური, რომ პროექტი არ არის მხოლოდ "მუშა სკრიპტი", არამედ დასრულებული პროდუქტი, რომელსაც გააჩნია საკუთარი ეკოსისტემა და პრეზენტაციის კულტურა.
